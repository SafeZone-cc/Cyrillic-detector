# Детектор кириллицы

## Как пользоваться:

1) обзавестить Microsoft Excel.
2) [включить макросы](http://safezone.cc/threads/kak-razreshit-zapusk-makrosov-v-microsoft-office-2003-2013.23152/)
3) ввести в ячейку F1 (самая верхняя на голубом фоне) нужное слово.
Профит.
4) или нажать "Проверка папки"... дальше каждое имя файла кликабельно.

ASC-коды символов также подсвечиваются в таблице символов.
Макрос написан поверх старой поделки кодировщика, поэтому в коде и на других листах много чего лишнего.
